# Math Flash Cards

A simple, fun flash card application for practicing addition and subtraction up to 20.

## Features

- **Addition & Subtraction**: Practice both operations up to 20
- **Multiple Modes**: Untimed practice or timed tests (1, 3, 5, or 10 minutes)
- **Vertical Math Layout**: Traditional flashcard style with stacked numbers
- **Fun Feedback**: Encouraging messages and sound effects for correct/incorrect answers
- **Progress Tracking**: Score, streak, and accuracy tracking
- **Gamification**: Earn awards based on performance
- **Session Reports**: Download detailed reports of your practice sessions

## Tech Stack

Pure vanilla JavaScript, HTML, and CSS - no frameworks or dependencies!

- Sound effects generated with Web Audio API
- Completely self-contained and works offline

## Usage

Simply open `index.html` in any modern web browser and start practicing!
