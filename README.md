# Math Flash Cards 🧮

A fun, interactive web-based flashcard game for practicing addition and subtraction up to 20. Built with pure vanilla JavaScript, HTML, and CSS - no frameworks or dependencies required!

## ✨ Features

### 🎯 **Multiple Game Modes**
- **Practice Mode**: Unlimited time to practice at your own pace
- **Timed Tests**: 1, 3, 5, or 10-minute challenges
- **Customizable Operations**: Choose addition, subtraction, or both

### 🎨 **Engaging User Experience**
- **Traditional Flashcard Layout**: Vertical math equations with stacked numbers
- **Real-time Feedback**: Encouraging messages and sound effects
- **Visual Progress Tracking**: Score, streak, and accuracy monitoring
- **Award System**: Earn badges for achievements and milestones

### 📊 **Comprehensive Analytics**
- **Session Statistics**: Detailed performance metrics
- **Downloadable Reports**: Export your practice sessions
- **Question History**: Track every answer and improvement
- **Achievement System**: Unlock awards based on performance

### 🔊 **Audio Feedback**
- **Web Audio API**: Generated sound effects for correct/incorrect answers
- **No External Dependencies**: All sounds created programmatically

## 🚀 Quick Start

### Option 1: Direct Play
1. Open `index.html` in any modern web browser
2. Select your preferred operations (addition/subtraction)
3. Choose a game mode and start practicing!

### Option 2: Local Development
```bash
# Clone the repository
git clone https://github.com/your-username/math-flashcards.git

# Navigate to the project directory
cd math-flashcards

# Open in your browser
open index.html
# or
python -m http.server 8000  # Then visit http://localhost:8000
```

## 🎯 How to Play

1. **Choose Operations**: Select addition, subtraction, or both
2. **Pick Your Mode**: 
   - Practice Mode for unlimited time
   - Timed tests (1, 3, 5, or 10 minutes)
3. **Solve Equations**: Enter your answer and press Enter or click Submit
4. **Track Progress**: Watch your score, streak, and accuracy improve
5. **Earn Awards**: Unlock achievements based on your performance
6. **Download Reports**: Save detailed session statistics

## 💝 Origin Story

This project was born from a simple desire: to help my son practice math in a fun, engaging way. I wanted to see how quickly we could build a complete educational game using modern AI tools like Claude and Cursor. The result? A fully functional flashcard game built in record time that my son absolutely loves!

The game combines the timeless appeal of traditional flashcards with modern web technologies, creating an interactive learning experience that keeps kids engaged while they practice essential math skills.
